# Descripción del trabajo

En este repositorio se recoge el desarrollo de una página web personal que he realizado como trabajo individual para la asignatura fundamentos de ingeniería informática del grado en ingeniería informática en la Universidad Francisco de Vitoria.
El objetivo principal de este proyecto ha sido poner en práctica los conocimientos básicos adquiridos sobre HTML y CSS, creando un sitio web estructurado y visualmente coherente que muestre información personal, académica y algunos temas de interés.
El sitio web está formado por varias secciones, organizadas en diferentes páginas HTML:

- Página principal (index.html)
- Sección acerca de mí (about.html)
- Descripción del grado universitario (degree.html)
- Página dedicada a la asignatura fundamentos de ingeniería informática (fii.html)
- Tema de interés personal (topic.html)
- Página de contacto (contact.html)
- Red de compañeros con enlaces a sus páginas web (net.html)

## Problemas durante el desarrollo

Uno de los principales problemas durante el desarrollo del proyecto ha sido entender cómo enlazar correctamente las distintas páginas HTML entre sí, especialmente al estar algunas dentro de la carpeta public.
También he tenido dificultades iniciales para conseguir que el diseño se viera de forma similar en todas las páginas, ya que he tenido que aprender a utilizar correctamente un único archivo CSS externo y aplicar las mismas clases en cada página.

## Conclusiones

La realización de este trabajo me ha permitido adquirir una base sólida sobre el uso de HTML y CSS para el desarrollo de páginas web sencillas.
Además, el proyecto me ha ayudado a comprender la importancia de una buena estructura del proyecto, la organización del código y el uso de git y github como herramienta de control de versiones y publicación de sitios web.
